# 內部專用中繼資料

metadata.md 檔案包含存放庫層級的中繼資料，會將以上資料傳遞至存放庫中的使用者指南 TOC.md 檔案。 若您想針對任何使用者指南，變更 metadata.md 的內容，請開啟任何 TOC.md 檔案，就能變更資料。

| 中繼資料 | 功能說明 |
|--- |--- |
| solution-title | 可用來當作文章頁首中的連結 |
| solution-hub-url | 開啟 helpx 說明中心頁面 |
| solution-icon | 請在解決方案標題旁，顯示解決方案圖示。 尚未執行 |
| getting-started-url | 連結至 HelpX 快速入門頁面 |
| tutorials-url | 連結至影片教學課程，包含 HelpX 教學課程，或是 KT 教學課程 |
| mini-toc-levels | 決定顯示在右側欄的標題層級數量。 預設值為 2 |
| git-repo | 指定供內部使用的主要存放庫位置 |

在 TOC.md 檔案中

| 中繼資料 | 功能說明 |
|--- |--- |
| user-guide-title | 可用來當作文章頁首中的連結 |
| user-guide-url | 開啟 helpx 說明中心頁面 |
