# 建立工作流程 {#how-to-create-a-workflow-in-adobe-campaign-standard}

工作流程可用於不同的內容，例如：

* 鎖定目標以管理對象或傳送訊息
* 資料管理(ETL)以處理資料
* 將資料匯入Campaign資料庫
* 技術流程，例如資料庫清理、復原追蹤資訊等。

此影片將示範如何在Adobe Campaign Standard中建立和編輯工作流程。
