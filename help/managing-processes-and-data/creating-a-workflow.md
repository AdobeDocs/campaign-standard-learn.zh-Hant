# 建立工作流程 {#how-to-create-a-workflow-in-adobe-campaign-standard}

工作流可用於不同的上下文，例如：

* 定位以管理對象或傳送訊息
* 資料管理(ETL)以控制資料
* 將資料匯入促銷活動資料庫
* 技術流程，例如資料庫清理、追蹤資訊的復原等。

此影片將示範如何在Adobe Campaign Standard建立和編輯工作流程。
