# 建立[!UICONTROL fatigue rules]

[!UICONTROL Fatigue rules] 可讓行銷人員設定全域跨通道業務規則，自動排除促銷活動中過度徵求的個人檔案。要建立[!UICONTROL fatigue rules]，您需要定義每個配置檔案的最大消息數，並選擇將應用規則的期間。 在準備傳送期間，設定檔會根據已傳送給設定檔的訊息數量，從傳送中排除（如果適用）。
