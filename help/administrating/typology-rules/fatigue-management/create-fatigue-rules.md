# 建立 [!UICONTROL fatigue rules]

[!UICONTROL Fatigue rules] 可讓行銷人員設定全球跨管道業務規則，自動排除行銷活動中過度請求的設定檔。
若要建立 [!UICONTROL fatigue rules]，您必須定義每個設定檔的訊息數上限，並選取將套用規則的期間。 在準備傳送期間，設定檔會根據已傳送給設定檔的訊息數量，從傳送中排除（如果適用）。
