# 建立 [!UICONTROL fatigue rules]

[!UICONTROL Fatigue rules] 可讓行銷人員設定全球跨頻道商業規則，自動從行銷活動中排除過度請求的設定檔。
建立 [!UICONTROL fatigue rules]，您需要定義每個設定檔的最大訊息數量，並選取將套用規則的期間。 在準備傳送期間，設定檔會根據已傳送給設定檔的訊息數量，從傳送中排除（如果適用）。
