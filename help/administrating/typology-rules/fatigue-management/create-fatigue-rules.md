# 建立 [!UICONTROL fatigue rules]

[!UICONTROL Fatigue rules] 允許營銷人員設定全局跨渠道業務規則，這些業務規則將自動從市場活動中排除過度請求的配置檔案。
建立 [!UICONTROL fatigue rules]，您需要為每個配置檔案定義最大消息數，並選擇規則將適用的期間。 在準備傳送期間，設定檔會根據已傳送給設定檔的訊息數量，從傳送中排除（如果適用）。
