# Adobe Campaign Standard快速入門

Adobe Campaign Standard是功能強大的跨通道宣傳管理工具，可協助您調整線上和線下策略，以建立個人化的客戶體驗。

控制建立、執行和追蹤跨通道宣傳的整個程式，並根據客戶的個人檔案、行為、偏好和需求傳送個人化訊息。 您可以輕鬆協調、建立模型並自動化整個行銷流程，並與其他Adobe解決方案整合。
