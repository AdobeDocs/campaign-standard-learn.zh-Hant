# 管理促銷活動

Adobe Campaign可讓您規劃行銷宣傳，以便建立和管理不同類型的活動： 電子郵件、簡訊訊息、 [!UICONTROL push notifications]工作流程、登陸頁面。 這些促銷活動及其內容可收集到程式中。 此影片將示範如何在Adobe Campaign中建立和管理促銷活動。
