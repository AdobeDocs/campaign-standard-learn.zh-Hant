# 使用CNAME（測試版）委派子網域

>[!IMPORTANT]
>
> 使用 CNAME 進行子網域委派為測試版，且會經常更新及修改，恕不另行通知。

若要開始使用Adobe Campaign傳送電子郵件或發佈登錄頁面，您必須設定新的子網域。

瞭解如何使用「控制面板」中的 CNAME 來設定和提交子網域。
