# 使用 CNAME (beta) 委派子網域

>[!IMPORTANT]
>
> 使用 CNAME 進行子網域委派為測試版，且會經常更新及修改，恕不另行通知。

若要開始傳送電子郵件或發佈登陸頁面給 Adobe Campaign，您必須設定新的子網域。
