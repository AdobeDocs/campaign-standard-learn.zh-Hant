# 建立測試設定檔 {#creating-a-test-profile}

測試設定檔可讓您鎖定與定義之目標定位條件不相符的其他收件者。它們會新增至訊息的對象中，以偵測任何詐用收件者資料庫的行為，或確保電子郵件送達收件匣中。
