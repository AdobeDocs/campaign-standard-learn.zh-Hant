# 建立測試設定檔 {#creating-a-test-profile}

測試設定檔可讓您鎖定與定義之目標定位條件不相符的其他收件者。它們被添加到郵件的受眾中，以檢測收件人資料庫的任何欺詐性使用，或確保電子郵件到達收件箱中。
