# 建立測試設定檔 {#creating-a-test-profile}

測試設定檔可讓您定位不符合已定義定位准則的其他收件者。 它們會新增至訊息的讀者群，以偵測任何詐用收件者資料庫的行為，或確保電子郵件送達收件匣。
