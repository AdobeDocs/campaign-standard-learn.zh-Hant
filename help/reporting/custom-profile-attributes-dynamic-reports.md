# 根據自訂設定檔屬性建立報告

Adobe Campaign的[!UICONTROL Dynamic Reports]是彈性的自由格式環境，您可以在其中加入自訂設定檔屬性來探索資料。 它可讓您結合任何組合的維度、量度和時間範圍，並具備無限制的劃分和比較。

此影片說明如何根據自訂設定檔屬性建立報告。
