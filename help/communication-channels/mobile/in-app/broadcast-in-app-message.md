# 廣播應用程式內訊息

您可以廣播應用程式內訊息，鎖定行動應用程式的所有使用者(無論Adobe Campaign是否有其設定檔資訊)。

此類別中的一些相關使用案例包括：

* 將行動應用程式教學課程傳送至首次應用程式啟動器，或
* 新功能發行公告
