# 廣播應用程式內訊息

您可以廣播應用程式內訊息，針對行動應用程式的所有使用者(不論Adobe Campaign是否有其個人檔案資訊)。 若要廣播訊息，您在設定應用程式內訊息時必須使用&#x200B;**[!UICONTROL Target all users of a mobile app template]**。

此類別中的一些相關使用案例包括：

* 將行動應用程式教學課程傳送至首次使用的應用程式啟動器或
* 新功能版本的公告
