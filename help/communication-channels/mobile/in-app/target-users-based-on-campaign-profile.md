# 根據其[!DNL Campaign]設定檔定位使用者

您可以鎖定已知的行動應用程式使用者（那些已透過PII登入應用程式的使用者），並透過Adobe Campaign Standard中提供的CRM設定檔屬性個人化訊息。 此範本在支援跨管道協調使用案例中非常實用，因為您已鎖定其他管道（例如電子郵件或推播）的使用者，並根據其回應，您想要透過[!UICONTROL In-App]訊息與他們互動。
