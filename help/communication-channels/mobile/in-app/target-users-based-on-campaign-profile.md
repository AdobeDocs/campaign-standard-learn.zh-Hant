# 根據[!DNL Campaign]設定檔定位使用者

您可以透過Adobe Campaign Standard提供的CRM個人檔案屬性個人化訊息，鎖定已知的行動應用程式使用者（已登入具有PII的應用程式的使用者）。 此範本對於支援跨通道協調使用案例最有用，因為您已針對電子郵件或推播等其他通道的使用者，而且您想要透過[!UICONTROL In-App]訊息與他們互動。
