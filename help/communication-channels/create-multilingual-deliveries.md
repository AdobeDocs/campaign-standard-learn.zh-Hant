# 建立多語言電子郵件或簡訊傳送

Adobe Campaign可讓您簡化傳送多語言宣傳的作業複雜性。 電子郵件傳送和簡訊傳送都可設為多語言，讓您根據描述檔偏好的語言來傳送電子郵件或簡訊。

此影片說明如何根據電子郵件電子報範例中的範本建立多語言傳送。影片中展示的方法適用於電子郵件和簡訊傳送。
