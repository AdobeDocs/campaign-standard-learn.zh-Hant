# 建立 [!UICONTROL Direct Mail] {#create-a-direct-mail}

此 [!UICONTROL Direct Mail] 通道可讓您擷取分段的收件者清單，以傳送至機構或上傳至SFTP伺服器。

此影片說明如何設定 [!UICONTROL Direct Mail] 傳遞以及可將設定檔新增至所需的設定 [!UICONTROL Direct Mail] 傳遞。
