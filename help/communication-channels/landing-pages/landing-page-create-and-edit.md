# 建立和編輯著陸頁面 {#create-and-edit-a-landing-page}

Adobe Campaign可讓您建立線上表單的登陸頁面，這些表單可用來擷取受眾的資訊、提供服務訂閱、顯示資料以及擴充資料庫。 著陸頁面也可用於取得或更新現有的描述檔。
