# 建立及編輯登陸頁面 {#create-and-edit-a-landing-page}

Adobe Campaign可讓您建立登入頁面，這些線上表單可用來擷取對象資訊、提供服務訂閱、顯示資料並擴充資料庫。 登入頁面也可用於取得或更新現有的設定檔。
