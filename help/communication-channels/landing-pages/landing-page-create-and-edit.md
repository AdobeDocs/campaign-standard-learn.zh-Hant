# 建立及編輯登陸頁面 {#create-and-edit-a-landing-page}

Adobe Campaign可讓您建立登陸頁面，這些登陸頁面是線上表單，可用來擷取受眾的資訊、提供服務訂閱、顯示資料，以及擴充資料庫。 著陸頁面也可用於取得或更新現有的描述檔。
