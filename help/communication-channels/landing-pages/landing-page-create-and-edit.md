# 建立及編輯登陸頁面 {#create-and-edit-a-landing-page}

Adobe Campaign允許您構建登錄頁，這些登錄頁是線上表單，可用於捕獲您的受眾資訊、提供服務訂閱、顯示資料和增長資料庫。 登錄頁也可用於獲取或更新現有配置檔案。
