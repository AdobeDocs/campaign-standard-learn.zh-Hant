# 建立及編輯登陸頁面 {#create-and-edit-a-landing-page}

Adobe Campaign可讓您建立登錄頁面，這些登錄頁面是線上表單，可用來擷取對象的資訊、提供服務訂閱、顯示資料及擴充資料庫。 登錄頁面也可用來取得或更新現有的設定檔。
